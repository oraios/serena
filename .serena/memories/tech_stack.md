# Serena Tech Stack

## Programming Language
- **Python**: 3.11+

## Build & Package Management
- **uv**: Dependency management and execution.
- **poethepoet**: Task runner for project-specific commands.

## Core Libraries & Protocols
- **LSP (Language Server Protocol)**: For symbolic code analysis (via Solid-LSP).
- **MCP (Model Context Protocol)**: Exposing tools to AI models.
- **Pydantic**: Data validation and settings management.
- **Jinja2**: Template engine for prompts.
- **Flask**: Used for certain internal dashboard/GUI components.
- **Pytest**: Testing framework.

## Quality Assurance Tools
- **Mypy**: Static type checking.
- **Black**: Code formatting.
- **Ruff**: Fast Python linter and formatter.
- **Codespell**: Spell checking for the codebase.
