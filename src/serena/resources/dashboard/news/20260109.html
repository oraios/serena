New Type Hierarchy and Documentation Tools

January 9, 2026

We're excited to announce the introduction of powerful new tools for code navigation and understanding:

  • Type Hierarchy Tool: Navigate and explore type relationships in your codebase with ease. View superclasses, subclasses, and implementation hierarchies.
  • Documentation Tools: Access comprehensive documentation directly from your code, including API references, examples, and usage patterns.

These tools enhance Serena's code intelligence capabilities, making it easier to understand complex codebases and navigate large projects efficiently.
