Update Your Launch Commands!

April 1, 2026

Users who run Serena via uvx (i.e. most users) are encouraged to update their launch commands. Unfortunately, uvx does not honour the Python version that is specified in our project file (see uv GitHub issue), which can result in a random Python version being used when launching Serena.

Specifically, add the parameters `-p 3.13` at the beginning of your command to ensure that Serena is run with Python 3.13.
